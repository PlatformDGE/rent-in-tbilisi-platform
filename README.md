# Molecula CRM

Первый рабочий прототип CRM для агентства недвижимости Molecula / Rent in Tbilisi.

## Что есть в прототипе

- Главная страница `Molecula CRM`
- Разделы: Объекты, Агенты, Сделки, Собственники, Клиенты
- Таблица объектов
- Карточка данных объекта в таблице
- Форма добавления объекта
- Хранение объектов в `localStorage` браузера
- Современный SaaS/CRM-интерфейс на белом фоне

## Стек

- React
- Vite
- TypeScript

## Запуск локально

```bash
npm install
npm run dev
```

После запуска Vite покажет локальный адрес, обычно:

```bash
http://localhost:5173/
```

## Сборка

```bash
npm run build
```
