# Molecula CRM

Первый рабочий прототип CRM для агентства недвижимости Molecula / Rent in Tbilisi.

## Что есть в прототипе

- Экран локального входа без сервера
- Dashboard после входа
- Левое меню и страницы: Объекты, Сделки, Агенты, Собственники, Клиенты
- Фильтры объектов по району, типу сделки и статусу
- Поиск объектов по адресу
- Статистика объектов: всего, в работе, сдано, продано, архив
- Таблица и карточки объектов
- Отдельная страница объекта
- Добавление, редактирование и удаление объекта
- Страница агентов и карточка агента
- Структура команды Molecula с ролями и комиссией
- Темная и светлая тема
- Хранение объектов в `localStorage` браузера
- Современный SaaS/CRM-интерфейс

## Стек

- React
- Vite
- TypeScript
- React Router

## Запуск локально

```bash
npm install
npm run dev
```

После запуска Vite покажет локальный адрес, обычно:

```bash
http://localhost:5173/
```

В текущей конфигурации можно также открыть:

```bash
http://127.0.0.1:5173/
```

## Вход

Авторизация локальная, без сервера. На экране входа выберите роль:

- Администратор
- Оператор
- Агент

Сессия сохраняется в `localStorage`.

## Сборка

```bash
npm run build
```
