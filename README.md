# Rent in Tbilisi Platform

Чистый frontend-каркас платформы управления арендой. Интерфейс построен на React, TypeScript и Vite и не содержит демонстрационных данных или подключений к внешним сервисам.

## Запуск

Требуется Node.js 20 или новее.

```bash
npm install
npm run dev
```

Vite выведет локальный адрес приложения в терминале.

## Проверка сборки

```bash
npm run build
npm run preview
```

Автоматические тесты в первой версии ещё не настроены.

## Структура frontend

- `src/app` — конфигурация приложения и маршрутизация;
- `src/components` — общие компоненты;
- `src/layouts` — каркасы desktop- и mobile-интерфейса;
- `src/modules` — функциональные разделы;
- `src/domain` — бизнес-сущности и правила;
- `src/repositories` — контракты доступа к данным;
- `src/services` — прикладные сценарии;
- `src/styles` — токены и глобальные стили.

## Ограничения первой версии

Backend, Telegram, Google Sheets и парсеры не подключены. Разделы показывают честные пустые состояния до появления реальных источников данных.
