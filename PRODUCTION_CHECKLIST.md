# Production checklist

- [ ] GitHub Secrets добавлены
- [ ] Telegram workflow проходит
- [x] `telegram-top10.json` содержит реальные данные
- [ ] Telegram-ссылки открываются в целевых браузерах и сетях
- [ ] Карта отображает реальные координаты
- [x] Топ-10 не изменён визуально
- [x] Нет просмотров
- [x] Нет demo/mock данных
- [ ] Мобильное меню проверено на production
- [x] Отображается имя пользователя
- [ ] Тёмная тема проверена на production
- [x] Build проходит
- [ ] Все тесты проходят
- [x] GitHub Pages работает
- [ ] Release branch создана
- [ ] Production tag создан
- [ ] Собственный домен настроен
- [ ] HTTPS работает на собственном домене
- [x] SPA routing работает после refresh на GitHub Pages

Release branch и tag создаются только после выполнения всех обязательных пунктов, включая реальные координаты и проверку Telegram-ссылок вне ограниченной среды.
